Notifier

Clean notifications plugin written in TypeScript with no dependencies

View on GitHub Download .zip Download .tar.gz

Install

npm install typescript-notifier

Building

npm run build

Usage

var notifier = Notifier.setup({
    theme: 'default',
    position: 'top-right'
});

notifier.get('Hello World', {
    type: 'error',
    delay: '6000'
});
